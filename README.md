## About Me

I am a cybersecurity professional focused on SOC operations, Managed Detection & Response, threat hunting, incident response, malware traffic analysis, and cloud security monitoring.

My work and learning are centered around turning raw security telemetry into clear investigative findings, practical detections, and actionable remediation guidance. I enjoy building hands-on security labs, analyzing network and endpoint artifacts, and documenting repeatable workflows for analysts.

Current focus areas:

- SOC/MDR investigation workflows
- Threat hunting and detection engineering
- DFIR and malware triage
- Network traffic analysis with Wireshark, Zeek, Suricata, and Zui
- SIEM operations with Splunk, Microsoft Sentinel, Elastic, and Wazuh
- Cloud security monitoring for AWS and Azure
- Agentic AI and SOAR-assisted security operations
